# Excel Data Analysis & Interactive Dashboard Project

This project focuses on analyzing sales and customer data using Microsoft Excel to generate meaningful business insights and build interactive dashboards for decision-making.

## Project Overview

The dataset includes information related to:

* Orders and sales performance
* Products and categories
* Profit and discount analysis
* Customer and regional data
* Shipping and order trends

## Tools & Techniques Used

* Microsoft Excel
* Power Query for data cleaning and transformation
* Pivot Tables & Pivot Charts
* Power Pivot
* Interactive Dashboards
* Data Visualization
* Data Cleaning & Preparation

## Key Tasks Performed

* Removed duplicate and inconsistent records
* Handled missing values and formatted data
* Analyzed sales, profit, discounts, and customer behavior
* Created dynamic reports and dashboards
* Visualized KPIs and business trends for better insights

## Dashboard Features

* Sales & Profit Analysis
* Regional Performance Tracking
* Category & Sub-Category Insights
* Customer Segment Analysis
* Order & Shipping Performance Monitoring

## Objective

The goal of this project is to transform raw business data into clear, interactive, and actionable insights that support data-driven decision-making.
